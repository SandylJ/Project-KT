# Project Chimera (Web)

This repository hosts a single-page web game. The site is published via GitHub Pages from the `docs/` directory.

## Structure

- `docs/`
  - `index.html`
  - `styles.css`
  - `js/app.js`
  - `assets/` (optional, for images and other static files)

## Local development

Open `index.html` in your browser.
